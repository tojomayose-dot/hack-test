🔌 ENDPOINTS API CLÉS

POST /api/auth/donor-login
├─ Input: { phone, password }
└─ Output: { user: { _id, name, phone, bloodGroup, location, isAvailable, role } }

PATCH /api/donors/:id/availability
├─ Input: { isAvailable: true|false }
└─ Output: { donor: { ... } }

GET /api/donations/:donorId
└─ Output: [{ _id, donationDate, hospitalId, volume, status, ... }]

POST /api/donations
├─ Input: { donorId, hospitalId, donationDate, healthNote, amount, status }
└─ Output: { _id, ... }

📱 FLUX UTILISATEUR COMPLET

1. Accédez à http://localhost:5173/login-donneur
2. Entrez votre téléphone + mot de passe
3. Cliquez "Se connecter"
4. Redirection vers /donor-space
5. Consultez profil, historique, disponibilité
6. Activez/désactivez disponibilité (toggle)
7. Allez dans "Points de Collecte"
8. Sélectionnez un centre
9. Remplissez formulaire don (si ≥56 jours depuis dernier)
10. Confirmation + retour profil
