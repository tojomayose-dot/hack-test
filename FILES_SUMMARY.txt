├─ Solution: npm install lucide-react

├─ Check: MongoDB running?

└─ Solution: sudo systemctl start mongod

Problem: CORS error
├─ Check: CORS enabled in backend?
├─ Check: API URL correct in frontend?
└─ Solution: Verify app.use(cors()) in server.js
